Lesson 1: The Basics of the Web and HTML

  • <Bold text: <b> text </b>
  • Italic text (em or emphasis): <em>text</em>
  • Links (Anchor): <a href="source url">text to hyperlink</a>
    • Example: Udacity
  • Images: <img src="url" alt="text"> (this is a void tag, i.e. no closing tag as it doesn't have any content)
    • Example:
  • Break (new line): <br> (another void tag)
  • Paragraph (puts a box around the content): <p>content</p>

A break tag is an inline tag, i.e. it only contains content.

A paragraph tag is a block tag, i.e. it can have height and width.

Container Elements

You can add styles (CSS) to these tags.

<span>text</span> is inline
This tag provides a way to add a hook to a part of a text or a part of a document. (Source: w3schools)

<div> text </div> is block
This tag is very often used together with CSS, to layout a web page. (Source: w3schools)

Lesson 2: Creating a Structured Document with HTML

HTML, CSS, JavaScript

They are all languages that form the componenets of a website. HTML gives Sturcture, CSS gives Style, and JavaScript gives the Interactive Components.

CSS = Cascading Style Sheets

DOM (Document Object Model) = tree-like structure of a page; a cross-platform and language-independent convention for representing and interacting with objects in HTML (HyperText Markup Language and other markup languages). The nodes of every document are organized in a tree structure, called the DOM tree.
